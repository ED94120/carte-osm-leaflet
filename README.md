# carte-ign-geometrie-antennes

Utilitaire cartographique web utilisant **Carte Facile** et **MapLibre**.

[Accéder à l'application](https://ed94120.github.io/carte-ign-geometrie-antennes/)

https://ed94120.github.io/carte-ign-geometrie-antennes

## Description

Cette application permet :

- de rechercher une adresse ;
- d'afficher la carte correspondante ;
- de cliquer sur deux points :
  - **Point 1** : lieu où l'exposition à une antenne-relais est calculée ;
  - **Point 2** : lieu de l'antenne-relais ;
- de tracer un segment entre ces deux points ;
- de mesurer la distance entre ces deux points en mètres ;
- de récupérer leurs coordonnées géographiques :
  - latitude ;
  - longitude ;
  - altitude au niveau du sol.

L'application calcule également les déports à l'axe de l'antenne :

- en azimut ;
- en élévation.
