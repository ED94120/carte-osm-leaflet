# carte-osm-leaflet

Utilitaire cartographique web utilisant Leaflet et OpenStreetMap.

Lien de lancement du programme :
https://ed94120.github.io/carte-osm-leaflet/

Description :
Cette application permet de rechercher une adresse, d'afficher la carte correspondante, de cliquer sur deux points pour tracer un segment, de mesurer la distance entre ces deux points en mètres, et de récupérer leurs coordonnées latitude et longitude.
