# carte-ign-geometrue-antennes

Utilitaire cartographique web utilisant Leaflet et OpenStreetMap.

Lien de lancement du programme :
https://ed94120.github.io/carte-ign-geometrie-antennes/

Description :
Cette application permet de rechercher une adresse, d'afficher la carte correspondante, de cliquer sur deux points (point 1 = lieu où l'Exposition à une antenne-relais est calculé, point 2 = lieu de l'antenne-relais) pour tracer un segment entre les 2 points, de mesurer la distance entre ces deux points en mètres, de récupérer leurs coordonnées latitude et longitude ainsi que leurs altitudes au niveau du sol. 

L'application calcule aussi les déports à l'axe de l'antenne en azimut et en élévation.
