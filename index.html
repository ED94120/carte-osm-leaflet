Calcul des déports à l'axe de l'antenne

Si la case Altitude IGN est cochée, les champs d'altitude au sol sont remplis automatiquement au clic sur la carte : le Point 1 alimente le champ Altitude au sol Point 1 et le Point 2 alimente le champ Altitude au sol Point 2 / antenne. Si la case est décochée, l'utilisateur doit saisir manuellement les deux altitudes. Le mode choisi est verrouillé entre le Point 1 et le Point 2, puis de nouveau modifiable après Effacer mesure.
Cliquer sur un premier point pour définir le point de calcul (PT1). Cliquer sur un second point pour définir l'antenne (PT2). Le segment est analysé dans le sens PT2 vers PT1. Le cercle est centré sur le dernier point cliqué.
Le sélecteur flottant Cartes en haut à droite permet de choisir le fond et d'activer séparément les surcouches, notamment Cadastre et Courbes de niveau, cumulables l'une avec l'autre.
Le programme interroge plusieurs services de données en ligne. Un temps d'attente de quelques secondes peut être nécessaire lors du premier clic sur le Point 1 pour récupérer l'altitude au sol. Ensuite, une fois les services sollicités et les connexions établies, la récupération des informations est généralement beaucoup plus rapide.
Sources de données utilisées : Géoplateforme IGN (altimétrie, géocodage), Carte Facile / MapLibre pour les fonds et surcouches, OpenStreetMap via Overpass API et GeoJSON ANFR par défaut pour l'affichage des supports.
